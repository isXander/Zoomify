# Zoomify 2.14.6

Zoomify now targets the following Minecraft versions:

- 1.20.1
- 1.20.4 (also supports 1.20.3)
- 1.20.6 (also supports 1.20.5)
- 1.21.1 (also supports 1.21.0)
- 1.21.3 (also supports 1.21.2, 1.21.4, 1.21.5)
- 1.21.6 (also supports 1.21.7, 1.21.8)
- 1.21.9

## Changes

- Add 1.21.9 target
- Fix crash on 1.20 builds relating to YACL dependency constraint
